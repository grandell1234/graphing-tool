# graphing-tool
A visual graphing tool with an easy-to-use GUI.
### Requirements:
```
python3 -m pip install pygame
```
or
```
pip install pygame
```
### How-to-install:
```
git clone https://github.com/grandell1234/graphing-tool.git
```
### Enter folder:
```
cd graphing-tool
```
### Launch:
```
python graph.py
```
